# ToolX — Quick Start for New Users

**Last updated:** 2026-02-14  

## What is ToolX?
ToolX is a ticketing and job-monitoring interface used to view **Tickets**, run **Jobs**, and check **System Status**.

## First login
1. Open ToolX.
2. Click **Sign in with SSO**.
3. If you see `ERR-AUTH-401`, follow the troubleshooting guide:
   - `Troubleshoot_ToolX_AUTH401.md`

## Daily workflow
- Check **Tickets** for assignments
- Use **Jobs** to rerun a failed batch
- Use **Help → Run Connection Test** if the Dashboard looks stale
